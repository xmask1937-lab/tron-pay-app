# Tron Pay 惠旺丝滑原型

**功能完整 MVP**:
- Tron (TRX + USDT TRC20) + Solana 支持
- 真实感余额查询 + 发送 (秒到账演示)
- 扫码收款 (QR)
- 交易历史
- 暗黑模式 + 惠旺风格
- 简单 Node.js 热钱包后端

**运行**:
```bash
cd tron_pay_app
flutter pub get
flutter run
```

**后端**:
```bash
cd backend
npm install
node server.js
```

**注意**: 此为学习原型，仅演示。生产需安全审计、私钥管理、合规等。  
Enjoy the 丝滑体验! 🚀
